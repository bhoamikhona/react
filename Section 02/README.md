# Section 02: Part 01 Introduction - React Fundamentals

**About:** In this part 01 of the course we will learn all about React fundamentals. We will acquire foundational skills such as setting up a new React app, React components, props, JSX, how to handle events, and the fundamentals of state and state management in order to make your apps dynamic. Part 01 also includes tons of small but beautiful exercises and challenges so that you can hit the ground running with the super important fundamentals and start writing a lot of code right away. Let's dive right into it.
